# Memory Monitor Widget Plugin

This project was developed as a memory monitor widget for windows widgets.

### Internal Settings

```json
{
  "usage_font_size": 18.0,
  "usage_foreground": "#FFBC8F8F",
  "graphic_color": "#FF2E6E02",
  "graphic_timeline": 200.0
}
```
